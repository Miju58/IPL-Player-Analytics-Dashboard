# 📊 IPL Player Performance Dashboard - Vaibhav Sooryavanshi

## 🔥 Project Overview
This Power BI dashboard analyzes the IPL career performance of Vaibhav Sooryavanshi. It provides insights into batting performance, opponent-wise runs, dismissals, and boundary analysis.

---

## 📌 Key Metrics
- Matches Played: 23  
- Total Runs: 1028  
- Strike Rate: 217.34  
- Highest Score: 103  
- Total Fours: 81  
- Total Sixes: 96  

---

## 📊 Dashboard Features

### 1. Career Overview
- Total performance KPIs
- Matches, Runs, Strike Rate

### 2. Opponent-wise Analysis
- Runs scored against each IPL team

### 3. Dismissal Analysis
- Caught, Bowled, LBW, Stumped breakdown

### 4. Boundary Analysis
- Fours vs Sixes contribution

### 5. Season-wise Performance
- Runs trend over IPL seasons

---

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization
